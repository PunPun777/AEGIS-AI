# Limitations

## Data Limitations

- Uses AG News dataset (not domain-specific)
- Labels generated using heuristics
- Lacks real geopolitical depth

---

## Model Limitations

- Keyword dependency
- Weak contextual understanding
- Struggles with edge cases

---

## System Limitations

- No real-time data ingestion
- No geographic awareness
- No temporal analysis

---

## Deployment Limitations

- Model stored via Git LFS
- Not scalable for production
- No cloud deployment yet

---

## Evaluation Limitations

- High accuracy due to data leakage
- Not representative of real-world performance

---

## Mitigation Plan

- Use real datasets (GDELT, ACLED)
- Improve labeling strategy
- Introduce anomaly detection
- Add explainability layer
